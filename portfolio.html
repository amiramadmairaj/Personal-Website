Odyssey

At Odyssey, I held a pivotal position in the development of a React web application used by internally by Odyssey employees. This webapp utilized AWS Amplify, DynamoDB, Lambda, and API Gateways to create a full stack web application. My contributions encompassed the implementation of crucial features such as sign-in/authentication and continuous integration/continuous deployment (CI/CD). To enhance functionality, I extensively utilized AWS Lambda to facilitate OpenAI API calls triggered by user questionnaire submissions within the mobile app to create three possible AI generated itenaries for the Odyssey employee (travel planner) to choose from. DynamoDB served as the storage solution for user data and preferences, ensuring scalability and efficiency. Operating in accordance with agile methodologies, our team consistently achieved timely milestones, resulting in the delivery of a seamless travel planning experience tailored to the needs of remote workers.

Slumber

I co-developed Slumber, an iOS wellness app that seamlessly integrated with Apple Watch and HealthKit. With Swift and SwiftUI, we implemented real-time biometric data collection and leveraged CoreML to provide personalized sleep and exercise recommendations based off a machine learning model created with training data we collected. Slumber aimed to quantify sleep tracking and analysis, empowering users with tailored suggestions for a healthier lifestyle.

MNST

In my Fashion-MNIST Dataset Classification Project, I implemented and assessed the performance of diverse machine learning classifiers, including K-nearest neighbors, Logistic Regression, Neural Networks, and Random Forest, using sci-kit-learn. I meticulously fine-tuned their hyperparameters to optimize their performance on the Fashion-MNIST dataset. Through comprehensive evaluation using various metrics and visualizations, I successfully identified the most effective classifier for the task and produced informative graphs and charts that highlighted the differences between the classification methodologies. This project showcases my ability to accurately classify data and effectively visualize and compare the results of machine learning classifiers.

Web Scraper

I created a highly efficient web scraper that effectively extracted data from UCI subdomains. By tokenizing words and creating an inverted index, I developed the foundation for a sophisticated search engine. Using techniques such as hashing, Cosine similarity, and TF-IDF, I ensured the retrieval of unique and relevant information. The search engine also featured a sleek user interface, delivering accurate search results and demonstrating my expertise in web scraping and information retrieval.
